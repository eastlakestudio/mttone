# 需求规格：会议基本信息非模态编辑与关联

## 1. 会议基本信息修改
- **主题**：编辑框形式，直接修改，限制在 200 字符内。
- **时间**：通过点击触发图形化 Popover 弹窗选择，包含日期和时分。
- **地点**：支持手动编辑。
- **参会人员**：标签排布，支持点击移除，支持打字回车追加。

## 2. 交互规格（非模态侧边栏）
- 位于详情页右侧，宽度固定 `240px`。
- 导航栏右上角折叠按钮支持开启和收起状态的本地持久化或运行时状态绑定。
- 所有字段修改通过失焦 (Focus Loss) 或回车触发静默数据库 update，无显式保存按钮。

## 3. 说话人绑定规则
- “参会人员”列表充当“说话人”的可选名册。
- 当用户点击左侧转写气泡中的说话人时，除了打字外，支持点击下拉小三角，调出当前已添加的所有“参会人员”姓名列表，点击即可直接将当前气泡（以及该 Speaker 关联的所有片段）的名字一键重命名。
