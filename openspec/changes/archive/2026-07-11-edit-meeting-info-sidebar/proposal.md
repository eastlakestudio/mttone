# 变更提案：会议信息非模态修改与参会人智能关联 (edit-meeting-info-sidebar)

## 1. 目标 (Goal)
在会议回顾与处理页面 (`ReviewingView.swift`) 引入右侧属性检查器折叠面板 (Inspector Sidebar)，允许用户非模态地编辑会议基本信息 (主题、时间、地点、参会人员)。同时，利用参会人数据打通转译中“说话人姓名”的快捷关联绑定，减少打字成本。

## 2. 变更范围 (Scope)
- **数据存储**：扩展 SQLite 的 `meetings` 表，追加 `attendees` 字段用于持久化参会人。
- **回顾视图重构**：将 `ReviewingView` 改为左右双列，右侧为信息检查器面板，支持一键折叠，修改后自动失焦存盘。
- **说话人关联**：在转写气泡 `TranscriptBubble` 的说话人输入框旁提供下拉菜单，列出当前会议的参会人供一键映射。
